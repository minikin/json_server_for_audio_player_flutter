## Start server

```sh
node server.js
```

## Get DB

```txt
http://localhost:4000/db
```

## Get Images

```txt
http://localhost:4000/assets/images/1.png
```

## Get Audio

```txt
http://localhost:4000/assets/audio/bensound-erf.mp3
```
